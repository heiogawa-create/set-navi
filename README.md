# セットナビ

持っているワックスから作れる髪型と、髪質・理想に合う市販ワックスが分かるメンズヘアセット診断Webアプリです。

## 主な機能

- 「〇〇（芸能人）の髪形みたいにしたい」など、自然なメッセージから髪型を提案
- 今の髪・なりたい髪型の写真を、撮影・ファイル選択・貼り付けで追加
- 髪の長さ・髪質・毛量・仕上がりを使ったワックス相性診断
- 持っているワックスから作りやすい髪型を提案
- 市販ワックス6種の比較とAmazon商品検索リンク
- ワックスの基本の付け方を4ステップの実写で解説
- 楽天ビューティ・OZmallの美容室検索・予約アフィリエイト導線
- スマートフォン対応

> 現在のβ版では、写真はブラウザ内の参考プレビューです。診断ロジックは選択項目とメッセージを中心に動作します。画像解析APIは今後追加できます。

## ローカル起動

Node.js 22.13以上を使用します。

```bash
npm run install:ci
npm run dev
```

本番ビルド:

```bash
npm run build
```

## Amazonアソシエイト設定

AmazonアソシエイトのトラッキングID `hei5731-22` を初期値として設定しています。別のIDに切り替える場合は、`.env.example` を `.env.local` にコピーして変更してください。

```bash
NEXT_PUBLIC_AMAZON_ASSOCIATE_TAG=hei5731-22
```

価格・在庫は表示せず、Amazonの商品ページで確認する設計です。

## 美容室予約アフィリエイト設定

楽天ビューティをメイン、OZmallをサブとして美容室予約導線を掲載しています。提供済みの成果計測付きURLを初期値に設定済みです。今後リンクを差し替える場合は、次の環境変数を使用します。

```bash
NEXT_PUBLIC_RAKUTEN_BEAUTY_AFFILIATE_URL=https://...
NEXT_PUBLIC_OZMALL_AFFILIATE_URL=https://...
```

OZmallの広告には、A8.netから発行された100×60のクリエイティブとインプレッション計測タグも掲載しています。広告コードや計測用URLを変更する場合は、各ASPで再発行した値を使用してください。
