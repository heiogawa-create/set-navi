# セットナビ

持っているワックスから作れる髪型と、髪質・理想に合う市販ワックスが分かるメンズヘアセット診断Webアプリです。

## 主な機能

- 「松潤みたいになりたい」など、自然なメッセージから髪型を提案
- 今の髪・なりたい髪型の写真を、撮影・ファイル選択・貼り付けで追加
- 髪の長さ・髪質・毛量・仕上がりを使ったワックス相性診断
- 持っているワックスから作りやすい髪型を提案
- 市販ワックス6種の比較とAmazon商品検索リンク
- ワックスの基本の付け方を4ステップのアニメーションで解説
- スマートフォン対応

> 現在のβ版では、写真はブラウザ内の参考プレビューです。診断ロジックは選択項目とメッセージを中心に動作します。画像解析APIは今後追加できます。

## ローカル起動

Node.js 22.13以上を使用します。

```bash
npm run install:ci
npm run dev
```

本番ビルド:

```bash
npm run build
```

## Amazonアソシエイト設定

`.env.example` を `.env.local` にコピーし、AmazonアソシエイトのトラッキングIDを設定してください。

```bash
NEXT_PUBLIC_AMAZON_ASSOCIATE_TAG=your-tag-22
```

未設定時もAmazonの商品検索ページへ移動しますが、アソシエイトIDは付きません。価格・在庫は表示せず、Amazonの商品ページで確認する設計です。

## プライバシー

追加した写真はブラウザ内でのみプレビューし、サーバーへ保存しません。
